Registration & Questionnaire Issues

Can't start the questionnaire

  1. 1.Use `ok!register` in the designated registration channel
  2. 2.Check if you already have a profile with `ok!mydata`
  3. 3.Try clicking the 'Begin Questionnaire' button instead of using commands
  4. 4.Ensure the bot has proper permissions in the channel
  5. 5.Wait a moment if you see a cooldown message

Questionnaire progress was lost

  1. 1.Your progress is automatically saved - try `ok!register` again
  2. 2.The questionnaire will resume where you left off
  3. 3.If it starts over, use `ok!mydata` to check if you completed it
  4. 4.Contact an admin if the issue persists

Can't submit text answers

  1. 1.Click the 'Provide Answer' button to open the modal
  2. 2.Ensure your answer meets character limits
  3. 3.Don't use special characters that might break formatting
  4. 4.Try using the command version if buttons aren't working

Command Issues

Commands not working

  1. 1.Verify you're using the correct prefix: `ok!`
  2. 2.Check if you're on cooldown (wait 30-60 seconds)
  3. 3.Ensure you're in a channel where the bot can see messages
  4. 4.Try using commands in a different channel
  5. 5.Check if the bot is online (look for green status)

Getting 'Permission Denied' errors

  1. 1.Some commands only work in specific channels
  2. 2.`ok!leavegroup` only works inside Orbit channels
  3. 3.Registration commands need the registration channel
  4. 4.Try using the command in a general channel

Buttons and dropdowns not responding

  1. 1.Interactive elements timeout after 15 minutes
  2. 2.Use the equivalent command instead of buttons
  3. 3.Ask an admin to refresh the panel
  4. 4.Try refreshing Discord (Ctrl/Cmd + R)
  5. 5.Check if you have a stable internet connection

Matching & Orbit Problems

Not getting matched to an Orbit

  1. 1.Ensure your questionnaire is 100% complete
  2. 2.Matching happens weekly - be patient
  3. 3.Check if there are enough users for matching (minimum 4)
  4. 4.Verify you're not already in an Orbit with `ok!mydata`
  5. 5.Ask an admin when the next matching cycle runs

Can't see my Orbit channel

  1. 1.Refresh Discord completely (Ctrl/Cmd + R)
  2. 2.Check if you're looking in the right category
  3. 3.Verify you were matched by checking announcements
  4. 4.Try logging out and back into Discord
  5. 5.Contact an admin to verify your Orbit assignment

Want to leave current Orbit

  1. 1.Use `ok!leavegroup` inside your Orbit channel
  2. 2.Click the 'Depart This Orbit' button in the channel
  3. 3.You'll be eligible for the next matching cycle
  4. 4.Consider discussing issues with your Orbit first
  5. 5.Report problematic members instead if that's the issue

Orbit channel is empty/inactive

  1. 1.Increase facilitation level for more prompts
  2. 2.Start conversations yourself - be the change!
  3. 3.Use icebreaker prompts from the bot
  4. 4.Consider leaving and waiting for a new match
  5. 5.Report to admin if all members are inactive

Data & Privacy Concerns

Can't view my data

  1. 1.Use `ok!mydata` - it's always sent via DM
  2. 2.Check your DM settings allow messages from bots
  3. 3.Ensure you're not blocking the bot
  4. 4.Try the command in a different channel
  5. 5.Enable DMs from server members temporarily

Want to delete all my data

  1. 1.Use `ok!deletedata` command
  2. 2.Confirm deletion via DM when prompted
  3. 3.This action cannot be undone
  4. 4.You'll be removed from current Orbit
  5. 5.Re-register anytime with `ok!register`

Concerned about message recording

  1. 1.Set facilitation to 'None' for complete privacy
  2. 2.No messages are recorded at 'None' setting
  3. 3.Only Orbit messages are analyzed, never DMs
  4. 4.Use `ok!mydata` to see what's stored
  5. 5.Read our privacy guide for full details

Bot Performance Issues

Bot is slow or unresponsive

  1. 1.Check if you're on cooldown
  2. 2.Wait a few moments and try again
  3. 3.Check Discord's status page for outages
  4. 4.Try commands in a less busy channel
  5. 5.Report to admin if issue persists over 5 minutes

Getting error messages

  1. 1.Screenshot the exact error message
  2. 2.Note what command/action caused it
  3. 3.Try the action again after a minute
  4. 4.Report to admin with details
  5. 5.Check if others have the same issue

Critical Issues

If you're experiencing critical issues that affect multiple users or system functionality:

  1. 1. Document the issue with screenshots
  2. 2. Note the exact time and commands used
  3. 3. Contact a server admin immediately
  4. 4. Admin will escalate to @skygorilla if needed
